# Forge - Landing page for a CNC manufacturer

Current design is a concept, made by me – [Andrew Shtyka](https://www.linkedin.com/in/andrewshtyka/), and [Danylo Dubrovskyi](https://www.linkedin.com/in/dubrovskydesign/) for [Lazarev Agency](https://www.linkedin.com/company/lazarevagency/) in Apr 2025. A year later I decided to transform a static design into an interactive webpage with React, Next.js and Motion.

Current project is a playground for improving my web-animation skills. Made for practice purposes only.

## Install

```bash
npm i
```

## Tech Stack
- Next.js 16 (App Router)
- React 19
- Motion (Framer Motion)
- Lenis
